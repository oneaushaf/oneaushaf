A little about PJC

PJC (pesona jati ciwangi) is a recreational site, located in ciwangi, a village in Garut, Jawa Barat. The site's main attraction is the pine forest with some beautiful panorama surounding

The PJC website is one of the project in my KKNT-i, developed in order to help the site and the vilage to gain more attention and credibility. It also provide a little ordering system to help the administrator manage orders.

*The app have been developed for mobile and desktop user, so feel free to use any device

Features and Capabilies

Facilities

The web will provide visitors with the facility provided in PJC. The display of the facility varies depending on the number of the image, wether or not it is orderable, and the text. Orderable facilities can be linked to the ordering calendar. The image number will affect the image will be displayed in carousel or not. It can be managed in the admin dashboard

Other Sites in The Surounding Village

The web also aims to increase visitor to ciwangi village, hence will also increase the possibility of visitors visiting PJC. Therefore, i created a section filled with surounding sites that might attract people to come to ciwangi. The Display would depend on the image, the existence of link or contact, and the index (odd or even). It can also be managed in the admin dashboard

Order

Some facility is able to be ordered. So we provide the orderable facilities with an ordering form.

To help visitors doing orders, i provide a calender for each orderable facilities so that they can automatically fill the form's facility and date and track the date that has been booked by others.

Admin

I make it so that the admin can be changed freely, but only one admin can exist so no admin addition is made. so the change of administrator and contact would not be a problem for the website. The picture, the account, and password can be changed anytime.
